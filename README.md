# homebrew-tap

`tree` for certificate chains. `man` pages for your projects.

## Usage

```
brew tap timorunge/tap
```

## Available formulae

### certree

Analyze and visualize X.509 certificate chains.

```
brew install timorunge/tap/certree
```

### lore

Search and serve your knowledge base from the terminal and over MCP.

```
brew install timorunge/tap/lore-cli
```

## Upgrade

```
brew update && brew upgrade
```
